# React-typescript-starter

Simple React boilerplate.

## feature

- [-] jest
- [-] offline
- [-] typescript
- [-] eslint
- [-] SSR

## get started

```
git clone https://github.com/kmkzt/react-typescript-starter myproject
cd myproject
yarn
yarn dev
```

## spa build

```
yarn build
```

## ssr development

```
yarn ssr:dev
```
